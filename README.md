# Cloud-Based Web Application Deployment

## Project Overview
This project demonstrates deployment of a simple web application using cloud-based hosting.

The application is deployed using GitHub Pages and can be accessed through a public URL.

## Technologies Used
- GitHub
- GitHub Pages
- HTML

## Deployment Steps
1. Created a GitHub repository
2. Developed a simple web application using HTML
3. Uploaded the code to GitHub
4. Enabled GitHub Pages for cloud deployment
5. Verified the live website
